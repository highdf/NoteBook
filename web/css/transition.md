### transition

#### transtion-property
`transtion-property`属性可以指定目标元素的哪些属性的值发生改变时，为这变化增加过渡交往果
```css
transition-property: 
    none |
    all |
    propertly-name 
```

#### transition-duration
`transition-duration`属性的作用是为拥有值变化过渡效果的元素设置过渡时长。
```css
transition-duration:
    <time> 


<time>:
    number + 's' |
    number + 'ms'
```
`transition-timing-function`定义了在值过渡的过程中属性值与时间的变化关系
```css
transition-timing-function:
    ease |
    ease-in |
    ease-out |
```
