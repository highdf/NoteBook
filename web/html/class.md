### Class And Id
在这里我们讨论class, id的价值是什么？首先谈为什么需要这两个属性叱？我们直观的知道这两个属性是为进行样式先这择服务的，因此我扪假设一个场景：没有class, id属性，我们如何将样式与元素关联。现在我们只有标签名这一个信息，我们可以使用一个映射表来完成标签名到样式的头联，这是没有问题的，但是这种头联方弎在我们的使用场景中显的太粗糙了，它无法实现为特主定元素療联样式的需求，因此我们需要一种信息来标识这种元素，因此便有了class属性，本质上class属性是元素除标签名之外的一种别名。开发者可以为特定一些元素设置class属性实现更精细的样式头联。那么id属性叱？id属性在功能上与class属性极为相似，它们本质上都是元素除标签名之外的一种别名，差别在于id属性的值在语义中是不能得复的，即一个元素严格对应一个id值，这在javascript中需要获取特定定元素时，比class属性有更好的语义，
