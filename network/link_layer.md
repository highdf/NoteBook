#### 数据链路层的作用是什么

#### 链路层的三个基本功能
| 功能 | 行为 | 目的 |
| --------------- | --------------- | --------------- |
| 数据封装 | 将网络层下放的数据添加起始与结束标识符 | 为接收端提供识别比特流中的有效数据的信息 |
| 透明传输 | 对原始数据内的特殊信息进行转义（如起始/结束标识符） | 解决原始数据内特殊字符造成的歧义 |
| 差错检测 | 在发送端生成检验码，在接收端验证检验码 | 解决数据单元出现比特钽误 |


##### 为什么鏈路层中对IP数据包的大小有限制，如果IP层下放的IP数据包过大，鏈路层一般会如何处理？
- 因为物理链路中实际传输的是链路层的数据单元帧，因此原始数据的大小受到硬件缓冲区大小，物理链路的传输能力的影响。
